1) Header & Tagline

My Personal Site

Nunquam Cedo — Never Surrender

The header contains the site title and #tagline. The CSS gives it a navy background and a lighter band below for emphasis.

<header> <h1>My Personal Site</h1> <p id="tagline">Nunquam Cedo — Never Surrender</p> </header>

2) Navigation (Accessible & Highlighted)

Home Hobbies Chess Science GiveMeStyle

Navigation uses Flexbox to space links evenly. aria-label="Primary" names the menu for accessibility. The active page uses aria-current="page" for highlight.

<nav class="site-nav" aria-label="Primary"> <a href="..." aria-current="page">Home</a> ... </nav>

9) Background Tile

The site background uses a tiled image with a fallback color, creating texture without distraction.

body{ background-image:url("../images/tile-bg.png"); background-repeat:repeat; background-attachment:fixed; background-color:#eef3fb; }
